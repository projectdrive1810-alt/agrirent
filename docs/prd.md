# AgriRent - Agricultural Machinery Rental Platform

## Problem Statement
Small farmers often cannot afford agricultural machinery such as tractors and harvesters. Machinery owners have idle equipment that could be rented out.

## Project Goal
Create a web platform that allows machinery owners to list equipment and farmers to rent it.

## Users
### Farmer
- Register/Login
- Search machinery
- Request bookings
- View booking status

### Machine Owner
- Register/Login
- Add machinery
- Manage bookings
- Accept/Reject requests

### Admin
- Manage users
- Manage machinery
- Monitor bookings

## Core Features
- Authentication
- Machine Listings
- Search & Filter
- Booking System
- Price Calculator
- Dashboard

## Tech Stack
- React
- Spring Boot
- PostgreSQL
- JWT Authentication
